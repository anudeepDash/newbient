# Future Plans & Roadmap - NewBi Entertainment

This document tracks upcoming features and ideas for the platform. Feel free to edit and add more items as they come up.

## 1. Event RSVP System
- [ ] **Email Invitations**: System to send automated RSVP emails to invitees.
- [ ] **RSVP Tracking**: Dashboard to track who has accepted, declined, or is tentative.
- [ ] **Status Updates**: Ability for users to update their RSVP status.

## 2. Digital Entry & Ticketing
- [ ] **QR Code Generation**: Automatically generate unique QR codes for confirmed attendees.
- [ ] **Check-in System**: Admin scanner (mobile view) to scan QR codes at the venue for seamless entry.
- [ ] **Ticket Validation**: Prevent duplicate entries and validate ticket authenticity.

## 3. Forms & Registrations
- [ ] **Volunteer Registration**: Dedicated form for people to sign up as volunteers for events.
- [ ] **Guest List (GL) Registration**: Form for promoters and partners to submit Guest List entries.
- [ ] **Vendor Applications**: Form for food, beverage, and merchandise vendors to apply.

## 4. Admin Enhancements
- [ ] **Concert Manager**: Fully implement the backend interactions for the Concert Manager (currently "Coming Soon").
- [ ] **Analytics Dashboard**: More detailed stats on ticket sales, revenue, and site traffic.

## 5. User Engagement
- [ ] **Photo Booth Uploads**: Allow attendees to upload their own event photos to a moderation queue.
- [ ] **Live Polls**: Real-time polling feature for events (e.g., "Next Song" requests).
